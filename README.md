
**Steps for every question:**

1. рҹҢұ Create a new branch, named after the question.
2. рҹ§© Solve it вҖ” write `ans.py` and its `README.md` on that branch.
3. рҹ’ҫ Commit using a consistent, dated format:
   ```
   dd-mm-yyyy Question_Name
   ```
4. рҹ”Җ Merge the branch back into `main`.

This keeps every problem's history isolated on its own branch before it joins `main` вҖ” good practice for branching, committing, and merging with real intent, not just habit.
